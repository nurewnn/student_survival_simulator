1. Problem Statement & Target Users

University life imposes dynamic, interrelated pressures on students. Academics, sleep deprivation, financial stability, and mental health are heavily dependent on each other, yet traditional student evaluation models are strictly binary or linear (e.g., "you are either failing or passing," "you are either stressed or calm").

Target Users: University students managing heavy semesters, academic counselors tracking student burnouts, and educational administrators optimizing curriculum design.

2. Why Fuzzy Logic is Appropriate

Human states like "feeling cooked," "being tired," or "being broke" are fundamentally subjective and vague. There is no precise threshold where a student suddenly transitions from "doing fine" to "completely burnt out." Fuzzy logic excels at uncertainty modeling by mapping these crisp data values to degrees of truth, allowing the simulator to run complex, multi-factor assessments simultaneously and return a smooth, defuzzified "Student Survival Score."

3. Fuzzy Variable Definitions

Input 1: Assignment Load (0 - 100) Sets: Light [0,0,20,45] | Packed [30,50,50,70] | Overloaded [55,80,100,100]
Input 2: Sleep Hours (0 - 12) Sets: Zombie [0,0,3,5] | Functioning [4,6,7,9] | Well-Rested [8,10,12,12]
Input 3: Wallet Balance (0 - 100) Sets: Broke [0,0,15,35] | Surviving [25,45,55,75] | Stable [65,80,100,100]
Input 4: Stress Level (0 - 100) Sets: Chilling [0,0,25,45] | Concerned [35,55,65,80] | Screaming [70,85,100,100]

4. Firing Rules Base (81 Rules)

Our system implements a comprehensive 81-rule base covering every combination of inputs. An output score is assigned based on a mathematical scoring of the linguistic combinations.
